# SNSを作りたい！

CMSの構築はWebエンジニアの基礎であり、最近の殆どのサービスはその延長としてSNSに落とし込まれます。<br>
ユーザー同士の評価やリプライによってデータに重みづけをし、内容によってサジェストを表示する仕組み。<br>
これらは既にネット世界の"普通"として受け入れられており、これを設計・構築することこそがWebエンジニアの仕事と言えるでしょう。<br>

## とはいえ..

業務の流れで結果的にSNSを作っていたことはあっても『最強のSNS』を考えて作った経験は無し。<br>
User / Category / Tag / Post / Comment モデル構成の基礎的なものを作っても面白く無いので、ここらで一発『最強のSNS』ってやつを作ってみようと思い立ちました。<br>
やるからにはGoogleを脅かす勢いでやってやろうと思ってます！（真顔）<br>
